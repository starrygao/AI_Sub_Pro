# 第三方声明

语言：[English](THIRD_PARTY_NOTICES.md) | [简体中文](THIRD_PARTY_NOTICES.zh-CN.md)

本仓库内置了压缩后的 Alpine.js 浏览器构建文件：
`app/static/vendor/alpinejs/alpine.min.js`，用于让本地界面不依赖 CDN 运行。

- Alpine.js：MIT License，https://github.com/alpinejs/alpine
- Tailwind CSS：MIT License，https://github.com/tailwindlabs/tailwindcss

Python 和 JavaScript 包依赖记录在 `requirements.txt`、`requirements-dev.txt`、
`package.json` 和 `electron/package.json` 中。
